# Telegram Clone Project 📱

> Um aplicativo de mensagens móvel construído com React Native e Expo

## 📂 Estrura do Projeto

Este repositório contém o projeto do Telegram Clone. O código principal está localizado no diretório `telegram-clone/`.

```
telegramclone/
├── README.md              # Este arquivo
├── package.json           # Dependências raiz
└── telegram-clone/        # Projeto principal
    ├── README.md          # Documentação detalhada do projeto
    ├── App.tsx            # Componente principal
    ├── src/               # Código fonte
    └── ...                # Outros arquivos de configuração
```

## 🚀 Quick Start

Para executar o projeto:

1. Acesse o diretório do projeto:
```bash
cd telegram-clone
```

2. Instale as dependências:
```bash
npm install
```

3. Inicie o servidor de desenvolvimento:
```bash
npm start
```

4. Escaneie o QR Code com o Expo Go no seu celular ou use um emulador.
